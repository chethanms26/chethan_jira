# User Guide

## Getting Started

### Account Creation
1. Visit the homepage
2. Click "Sign Up"
3. Fill in your details
4. Choose account type (Artisan/Buyer)

### For Buyers
- Browse products
- Add items to cart
- Place orders
- View order history
- Download invoices

### For Artisans
- List new products
- Manage inventory
- View sales reports
- Update product details
- Process orders
